# 📊 Análise de Vendas no Excel

Projeto simples desenvolvido para praticar Excel básico.

## 📌 Funcionalidades
- Tabela de vendas
- Cálculo automático (Quantidade × Preço)
- Faturamento total
- Média por venda
- Maior e menor venda
- Soma por produto (SOMASE)
- Gráfico de faturamento

## 🧠 Aprendizados
- Uso de fórmulas (SOMA, MÉDIA, MÁXIMO, MÍNIMO, SOMASE)
- Organização de dados
- Criação de gráficos
