# 📊 Dashboard de Vendas no Excel

Projeto de análise de vendas desenvolvido em Excel, utilizando tabelas dinâmicas, gráficos e segmentação de dados para criação de um dashboard interativo.

---

## 📌 Funcionalidades

- Tabela de vendas estruturada
- Cálculo automático (Quantidade × Preço)
- Faturamento total
- Média por venda
- Maior e menor venda
- Soma por produto (SOMASE)
- Tabela dinâmica
- Segmentação de dados (filtros interativos)
- Dashboard interativo
- Gráfico de faturamento por produto

---

## 🧠 Aprendizados

- Uso de fórmulas (SOMA, MÉDIA, MÁXIMO, MÍNIMO, SOMASE)
- Estruturação de base de dados
- Criação de tabelas dinâmicas
- Criação de dashboards no Excel
- Visualização de dados com gráficos

## 🛠️ Tecnologias utilizadas

- Microsoft Excel

---

## 📂 Estrutura do projeto

- **Base_Dados** → dados brutos
- **Analises** → cálculos e tabela dinâmica
- **Dashboard** → visualização e interação

---

## 🚀 Objetivo

Praticar conceitos de análise de dados utilizando Excel, simulando um cenário real de acompanhamento de vendas.

---

## 📎 Como usar

1. Abra o arquivo `.xlsx`
2. Navegue até a aba **Dashboard**
3. Utilize os filtros para interagir com os dados
